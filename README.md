# vers

> Share your `--version`s with friends over the [cryptonet][ssb]

## Usage

###### List all your version

```sh
vers

 git  2.14.1
node  8.7.0
 npm  5.4.2
 ```

###### Share you version with peers over [ssb]

```sh
vers share

         git      node     npm

   alan  2.14.1   8.7.0    5.4.2
bernard  2.14.1   8.7.0    5.4.2
   rich  2.14.1   8.7.0    5.4.2
```
